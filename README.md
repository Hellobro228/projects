# Мои проекты

В папке CV можно найти интересные проекты по компьютерному зрению:
* **semantic_segmentation** — решалась задача сегментации родинок путем реализации SegNet
* **autoencoders** — автоэнкодер, который решает задачу реконструкции изображений, также можно менять лица людям: добавлять улыбку, закрывать глаза и тп
