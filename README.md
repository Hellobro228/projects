# Мои проекты

В папке **CV** можно найти интересные проекты по компьютерному зрению:

* **semantic_segmentation** — решалась задача сегментации родинок путем реализации SegNet
* **autoencoders** — автоэнкодер, который решает задачу реконструкции изображений, также можно менять лица людям: добавлять улыбку, закрывать глаза и тп

**NLP**:

* **classification_with_BERT** — классификация комментариев с помощью BERT
* **translation_RNN_attention_EN_DE** — RNN+attention переводчик с английского языка на немецкий 

**Classic ML**:

* **logreg_classification_text** —  классификация текстов с реализацией собственного токенеизатора и использованием логистической регрессии
* **trees** — реализация дерева решений с жадным построением и решение задачи классификации с помощью деревьев, бэггинга и случайного леса
